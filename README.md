## Description

Memory Game is an interactive web application where players flip cards to find matching pairs.

The project focuses on implementing game logic with TypeScript, structuring code in a modular way, and styling a responsive interface using SCSS. Vite is used as a modern build tool to enable fast development and optimized production builds.

The application demonstrates state management, DOM manipulation, event handling, and clean project architecture.

---

## Features

- Interactive card matching gameplay  
- Randomized card positions  
- Smooth flip animations  
- Move tracking  
- Win condition detection  
- Responsive layout  
- Modular TypeScript structure  
- Fast development with Vite  
- Production build optimization

## How to Run

Clone the repository and install dependencies:

```bash
git clone https://github.com/your-username/memory-game.git
cd memory-game
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```
